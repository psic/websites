Pour qui ? Pour tout le monde : artisans, commerçants, PME ou associaions qui souhaitent avoir un petit site web de présentations, particuliers ayant le désir de s'exprimer sur le web




L'hébergement gratuit, c'est quoi ?

Avoir une page web sur Internet c'est :
  • Des pages web, le plus simplement ce sont des documents html (et css) reliés entre eux,
  • un nom de domaine, c'est à dire une adresse internet pour votre site,
  • un hebergeur, c'est à dire un ordinateur relié à Internet, configuré pour mettre à disponition vos pages web à l'adresse choisie.

Je vous fournis gratuitement, pour l'instant, les deux derniers services. L'adresse internet de votre site sera quelque chose comme ça : votre_site.web-en-royans.fr, par exemple lebarajus.web-en-royans.fr. L'hergerment de votre site web sera réaisé sur un serveur, chez moi. Pour cette raison, il se peut que votre page (ainsi que ce site) soit parfois inaccessible quelques heures, ne serais que pendant ou après une coupure de courant. Il se peut que vous ayez besoin de technologies supplémentaires : php, ruby/rails, MySQL, etc... tout ceci devrait être disponible.

Pour la création de site web, je peux vous aider, vous guider, vous former ou même faire pour vous, mais ces prestations seront facturées.

Ces services sont en priorité reservés aux acteurs/habitants du pays du Royans et des communes environnantes.

Un annuaire web, c'est quoi ?

Le but est d'établir un liste hierarchisées des sites web des entreprises, commerçants, artisans, associations, particuliers, ... résidants dans le pays du Royans. Cet annuaire est en cours de création, et sera en constant évolution. N'hésitez pas à me contacter pour y figurer.
